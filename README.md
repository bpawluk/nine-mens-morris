# nine-mens-morris
WPF Nine Men's Morris game for one, two or zero players (AI vs AI) using MinMax and AlfaBeta algorithms

- **TECH STACK:** .NET WPF
- **TRY IT OUT:** [Mlynek v1.0.0](https://github.com/bpawluk/nine-mens-morris/releases/tag/v1.0.0)
